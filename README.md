## Master's Degree Thesis Project
Data and analysis from pilot study for Master's Thesis Project from the Universitat Pomepu Fabra, Barcelona, Spain. 

The thesis title is: Exploring Methods of Rapid Exposure in Virtual Reality for Public Speaking Anxiety. 
The code provided is poorly written and will be improved incrementally. It was the first time analyzing data using Python and Jupyter Notebooks for a complete project. 

The thesis manuscript can be found in the repository.
